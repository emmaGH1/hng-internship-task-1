Front-End Track - Stage One Task
Task Description
The objective of this task is to develop and host a web page using HTML, CSS, and JavaScript that showcases specific personal and real-time data attributes. Each required element on the webpage should have a specified data-testid attribute for easy identification and testing.

Requirements
1. Slack Name
Objective: Display your Slack name prominently.

Attribute: data-testid="slackUserName"

2. Slack Display Picture
Objective: Showcase your Slack profile picture.

Details: The alt attribute of the image should be set to your Slack username.

Attribute: data-testid="slackDisplayImage"

3. Current Day of the Week
Objective: Indicate the current day of the week (e.g., Monday, Tuesday).

Attribute: data-testid="currentDayOfTheWeek"

4. Current UTC Time
Objective: Display the current UTC time in milliseconds.

Attribute: data-testid="currentUTCTime"

5. Track
Objective: Mention your track (e.g., Frontend).

Attribute: data-testid="myTrack"

6. GitHub URL
Objective: Provide a clickable link to your GitHub repository where the source code for this task resides.

Attribute: data-testid="githubURL"

Task Completion
Please make sure to fulfill all the requirements listed above. Once completed, you can host your web page to make it accessible to others.
